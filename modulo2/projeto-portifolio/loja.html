Projeto Lojinha

Projeto desenvolvido para os primeiros contatos com HTML e CSS.

É uma página protótipo de um controle de estoque e de vendas de uma lojinha.

Neste projeto conhecemos sobre listas, tabelas, formulários, botões e links do HTML e algumas propriedades básicas de estilização do CSS.

Para criar as montagens de capas dos discos de vinil, usei de alguns conhecimentos básicos que possuo de manipulação de imagens através das ferramentas Gimp e Photopea.
